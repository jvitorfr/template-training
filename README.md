## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

## Instalação
Seja bem vindo ao Template-training
Oque é o Template-training? Um projeto em

- PHP
- MYSQL
- VUEJS

Para instação do mesmo, preste atenção no arquivo docker-compose.yml

configura os paths de acordo com os caminhos de sua maquina e execute o docker-compose up.

Verifique os erros e os ajuste.

Após ajustar os erros, pode-se utilizar docker-compose up -d (sem o debug).

Agora entre no container do php e realize a instalação das extensões de banco de dados para php.
Rode também as migrações de banco de dados

docker exec -ti php-container bash 
> docker-php-ext-install mysqli pdo pdo_mysql
> php artisan migrate:install
> php artisan migrate
> exit

Reinicei os containers:
docker-compose stop
docker-compoe up -d

acesse a rota de conexão.

localhost/test.php

Caso a mensagem de conexão seja apresentada, então seu laravel + mysql estão funcionando perfeitamente.

Após isso acesse a rota

localhost:8080

Caso a  tela inicial do VUE seja apresentada, então o seu template está pronto para iniciar o desenvolvimento.
